Поддержка изображений

Улучшенный API-парсер позволяет:

  • Извлекать текст книги со всех страниц
  • Сохранять изображения из книги
  • Создавать структурированный JSON с контентом
  • Автоматически перехватывать API-запросы

Например: https://read.amazon.com/?asin=B009SE1Z9E
Учетные данные необходимы для авторизации и доступа к книге
Максимальное количество страниц для извлечения
Время ожидания загрузки страницы
Результат также будет сохранен в JSON формате в файле с расширением .json
Все изображения из книги будут сохранены в этой директории

Как работает улучшенный парсер

                    

                    

                        

                            

                                

                                    🔍
                                

                                
Автоматическое извлечение

                                
Парсер самостоятельно авторизуется, открывает книгу и перехватывает все API-запросы, в том числе запросы к изображениям.

                            

                            
                            

                                

                                    🖼️
                                

                                
Сохранение изображений

                                
Парсер извлекает и сохраняет все изображения из книги, восстанавливая их из перехваченных blob-данных и base64-строк.

                            

                            
                            

                                

                                    📚
                                

                                
Пагинация и структура

                                
Автоматически выполняется пагинация по заданному количеству страниц, с сохранением структуры книги и номеров страниц.

                            

                        

                        
                        

                            Важно: 
                            
    
                                
  • Для работы парсера требуется валидный аккаунт Amazon с доступом к Kindle Cloud Reader.
    • 
                                
  • Необходимо указать корректный URL книги в формате Kindle Cloud Reader.
    • 
                                
  • Процесс может занять продолжительное время в зависимости от количества страниц и выбранной задержки.
